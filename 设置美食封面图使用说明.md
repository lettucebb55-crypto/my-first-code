# 设置美食封面图使用说明

## 📋 功能说明

将5张美食图片依次设置为美食的封面图。

## 🎯 使用步骤

### 步骤1：准备图片文件

将5张美食图片放到项目根目录下的 `media/food_images_to_add/` 目录中。

**推荐的图片文件名：**
- `food_1.jpg` - 保定驴肉火烧
- `food_2.jpg` - 白洋淀全鱼宴  
- `food_3.jpg` - 保定糖葫芦
- `food_4.jpg` - 其他美食
- `food_5.jpg` - 其他美食

**支持的图片格式：**
- `.jpg` / `.jpeg`
- `.png`
- `.gif`
- `.webp`

**目录结构示例：**
```
baoding_tourism/
├── media/
│   └── food_images_to_add/
│       ├── food_1.jpg
│       ├── food_2.jpg
│       ├── food_3.jpg
│       ├── food_4.jpg
│       └── food_5.jpg
```

### 步骤2：运行管理命令

```bash
python manage.py set_food_cover_images
```

命令会自动：
- 读取 `media/food_images_to_add/` 目录下的所有图片（按文件名排序）
- 将图片依次分配给美食（按display_order、热门、推荐排序）
- 设置封面图

### 步骤3：查看结果

访问美食列表页，即可看到设置好的封面图。

## 🔧 命令参数说明

```bash
python manage.py set_food_cover_images [选项]
```

**可用选项：**

- `--image-path PATH`：指定图片文件路径（可多次使用）
- `--from-dir DIR`：从指定目录读取图片（默认：`media/food_images_to_add`）

**示例：**

```bash
# 从默认目录读取图片
python manage.py set_food_cover_images

# 指定图片路径
python manage.py set_food_cover_images --image-path ./images/food1.jpg --image-path ./images/food2.jpg

# 从自定义目录读取
python manage.py set_food_cover_images --from-dir ./my_food_images
```

## 📝 注意事项

1. **图片顺序**：图片会按照文件名排序，依次分配给美食
2. **美食顺序**：美食按照 `display_order`、`is_hot`、`is_recommended` 排序
3. **图片数量**：
   - 如果图片多于美食，只处理前N张（N=美食数量）
   - 如果图片少于美食，只处理前N个美食（N=图片数量）
4. **图片大小**：建议图片大小不超过5MB

## 🎉 完成

设置完成后，刷新浏览器即可看到美食的封面图！

---

**提示**：如果您有实际的图片文件，请将它们放到 `media/food_images_to_add/` 目录，然后运行管理命令即可。
